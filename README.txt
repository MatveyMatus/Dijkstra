               *Использование*
для подсчёта примера нужно записать пример в файлик expr.txt.
И указать его вторым аргуметом командной строки. (node Dijkstra.js expr.txt)
          *Примечание для операции возведении в степень* 
Для выполнение операции (2+3)^2^3 = 390625 нужно записать следующим образом ((2)^(2^3)) = 390625
